## 🛠 Usage Guidelines

- **Serving:** To preview any prototype, ensure the directory is served via a local development server (e.g., `python3 -m http.server 8080`).
- **Dependencies:** Most prototypes use CDN-linked Tailwind CSS for portability.
- **Contributions:** If you are building a new experiment, clone the `prototype-template` and register it in this `README.md`.

## ⚙️ CI/CD & Deployment

- Git syncing is managed via internal M4 automation.
